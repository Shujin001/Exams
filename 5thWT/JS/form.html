      Context handling:
         - field's value check (direct/basic technique)
         - compare string pattern with field's value (REGEX)
      
      REGEX: Regular Expression (A string pattern)
         ex: \^([a-zA-Z]){3}$\g
             -----------------
              [a-zA-Z]  : support small and capital letter
              (3)       : limit of character in string
              \g        : global flag         
              $         : fixed end

         -- REGEX methods in JS:
                 pattern: var regex_pattern = /^([a-zA-Z]){3}$/
                 - fieldValue.match(regex_pattern)
                 - regex_pattern.test(fieldValue)

         -- Documents:
                 - regexr.com
                 - regex101.com   

         ex: email basic pattern:
               let emailRegex = /^([A-Za-z])+\w(@)[a-z]+\w+(.)[a-z]{3}$/g
               
               if(email.value.match(emailRegex)) {...}
               else {...}
